Copse

Five commands for managing agent-created PRs

Streamline your AI workflow

Copse helps you manage pull requests created by Cursor and Claude agents with simple, powerful commands.

✓

approval

Enable merge when ready on matching PRs

+

create-prs

Find recent agent branches and create PRs

◉

pr-status

View open agent PRs with test failures

↻

rerun-failed

Rerun failed workflow runs on branches

⇄

update-main

Merge main into PR branches to keep them current

Get Started

1

Install Prerequisites

You'll need GitHub CLI and Node.js 18+

2

Install Copse

npm install -g copse

Or clone and use directly:

git clone https://github.com/YOUR_USERNAME/copse.git
3

Start Using

approval owner/repo cursor

Enable auto-merge on all your Cursor agent PRs

Quick Examples

Enable merge when ready

approval acme/project cursor

Create PRs from recent branches

create-prs acme/project claude

Check PR test status

pr-status acme/project

Rerun failed tests

rerun-failed acme/project cursor

Update PRs with main

update-main acme/project

Preview without changes

approval acme/project cursor --dry-run
